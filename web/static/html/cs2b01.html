Desarrollo basado en plataforma

Resumen del curso:

Es el diseño y desarrollo de aplicaciones de software para una plataforma específica. Esta difiere de la programación de propósito general, ya que el desarrollo basado en plataforma toma en cuenta las limitaciones de cada plataforma. Estas plataformas se caracterizan por el uso de APIs especializadas, mecanismos de entrega / de actualización distintos, y se resumieron lejos del nivel de la máquina.

Las principales plataformas son las siguientes:

  • Plataforma tradicional
  • Plataforma web
  • Plataforma móvil
  • Plataforma en la nube
